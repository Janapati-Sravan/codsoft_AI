# Face Detection and Recognition System

## Introduction

The Face Detection and Recognition System is a Computer Vision and Artificial Intelligence project developed using Python, OpenCV, and InsightFace. The system is capable of automatically capturing face images, training a face recognition model, and identifying individuals in real-time through a webcam.

This project utilizes deep learning-based facial embeddings generated by InsightFace, providing significantly better accuracy than traditional face recognition methods.

---

* Detect human faces from live video streams.

## Dataset Collection

The project includes an automated dataset collection feature.

Run the following command:

```bash
python train.py --capture
```

### Steps

1. Enter the person's name.
2. Look directly at the webcam.
3. Press the Space key to capture images.
4. Move your head slightly between captures.
5. Continue until the required number of images is collected.

Images are automatically saved inside:

```text
dataset/
└── Person_Name/
```

By default, the system collects:

```python
PHOTOS_NEEDED = 50
```

images for each individual.

---

## Model Training

After collecting the dataset, train the recognition model using:

```bash
python train.py
```

### Training Process

* Loads all images from the dataset folder.
* Detects faces using InsightFace.
* Extracts facial embeddings.
* Associates embeddings with person names.
* Saves the trained model.

Output:

```text
face_model.pkl
```

The model file contains:

* Face embeddings
* Person labels
* Training information

---

## Running the Recognition System

Start the face recognition application using:

```bash
python main.py
```

The webcam will open and begin detecting and recognizing faces in real-time.

---

## Recognition Workflow

1. Capture frames from webcam.
2. Detect faces using InsightFace.
3. Generate face embeddings.
4. Compare embeddings with trained embeddings.
5. Calculate similarity scores.
6. Identify the best match.
7. Display the person's name and confidence score.

## Configuration Parameters

```python
SIMILARITY_THRESH = 0.35
MIN_MARGIN = 0.04
SMOOTH_FRAMES = 6
PROCESS_EVERY = 3
FRAME_WIDTH = 480
```

### Parameter Description

**SIMILARITY_THRESH**

Determines the minimum similarity score required for recognition.

**MIN_MARGIN**

Prevents incorrect predictions when two identities have very similar scores.

**SMOOTH_FRAMES**

Improves recognition stability by averaging predictions over multiple frames.

**PROCESS_EVERY**

Processes every nth frame to improve performance.
